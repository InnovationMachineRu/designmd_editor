# Scaffolding rules — Svelte 5

- Tooling: Vite + Svelte 5 + TypeScript.
- Component shape: runes (`$props`, `$state`, `$derived`, `$bindable`), typed props,
  semantic event dispatch.
- Tokens: CSS custom properties on `:root` from DESIGN.md; component `<style>` reads vars.
- Stories: Storybook (Svelte + Vite), one per component.
- Tests: Vitest + Svelte Testing Library + axe.

Follow `.claude/best-practices/frameworks/svelte.md`.
